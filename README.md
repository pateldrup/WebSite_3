🎓 APEUni Clone Website

A responsive APEUni (PTE Practice Platform) Clone built using HTML & CSS.
This project replicates the UI and layout of the APEUni website, focusing on clean design and user experience.

🌐 Live Demo

👉 https://steady-medovik-9e9c73.netlify.app/

📖 About APEUni

APEUni is an AI-powered platform used for preparing the PTE (Pearson Test of English) exam. It provides features like practice questions, AI scoring, and study materials for students.

🚀 Features
🏠 Clean Homepage UI
📚 PTE Study Sections (Speaking, Writing, Reading, Listening)
🎨 Modern UI Design
📱 Fully Responsive (Mobile + Desktop)
⚡ Fast Performance (Pure HTML & CSS)
🛠️ Tech Stack
HTML5
CSS3 (Flexbox + Grid)
